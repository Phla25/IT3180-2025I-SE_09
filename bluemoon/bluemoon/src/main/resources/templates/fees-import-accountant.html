Nhập Dữ Liệu Từ Excel

Tải lên file Excel (.xlsx) để tạo hàng loạt hóa đơn

Kéo thả file hoặc click để chọn

Cấu trúc file Excel yêu cầu (6 cột):

  • Cột A: Mã Hộ gia đình (Bắt buộc).
  • Cột B: CCCD Thành viên (Tùy chọn. Trống = Hóa đơn chung).
  • Cột C: Loại phí.
  • Cột D: Số tiền (VNĐ).
  • Cột E: Hạn thanh toán (dd/MM/yyyy).
  • Cột F: Ghi chú.
