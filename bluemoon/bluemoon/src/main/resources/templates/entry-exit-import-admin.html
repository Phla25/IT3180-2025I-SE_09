Cấu trúc file Excel yêu cầu (5 cột):
1. CCCD (Bắt buộc)
2. Họ và Tên (Nếu là người lạ chưa có trong hệ thống)
3. Ngày sinh (dd/MM/yyyy)
4. Thời gian (dd/MM/yyyy HH:mm)
5. Loại (VAO/RA) - Cổng

Nhấn để chọn file hoặc kéo thả vào đây
